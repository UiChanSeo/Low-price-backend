# 설명
```
- 이 앱은 Low-Price-App의 백엔드로 동작하는 프로그램입니다.
- Low-Price-App 사용자의 정보를 서버에 저장을 하거나, 서버에 저장된 정보를 앱으로 가져갈 떄 백엔드에서 역할으 합니다.
- 사용자 정보를 관리합니다.
```

# 실행
```
RUN="dev" PYTHONPATH=${PWD} gunicorn app.run_api:app -b 0.0.0.0:8080 -w 3
```
